## Setup

- SPI must be enabled

```bash
git clone https://github.com/williamluke4/TideTracker 
cd TideTracker
sudo ./install.sh
# "Uninstall"
sudo ./uninstall.sh
```

## Development

```bash
# Requirements
python3 -m pip install -r requirements.txt

# Running
python3 -m tidetracker.run
```
